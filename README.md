# ✈️ Safarnama – Travel Booking Website

Safarnama is a modern and responsive travel booking website built using **HTML, CSS, and JavaScript**. It provides a smooth user experience for exploring destinations, browsing flights and hotels, and completing bookings through an interactive frontend.

## 🌐 Live Demo

🔗 **Website:** https://safarnama-blue.vercel.app

## 📂 GitHub Repository

🔗 https://github.com/anushka17jaiswal/SAFARNAMA

## ✨ Features

- 🎥 Video background hero section
- 🏝️ Popular destinations (Kerala, Goa, Ladakh & Rajasthan)
- ✈️ Flight booking interface
- 🏨 Hotel booking interface
- 📝 Interactive booking form
- ✅ Booking confirmation page
- 💾 Local Storage support
- 📱 Fully responsive design
- ❤️ Modern Red & White UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/anushka17jaiswal/SAFARNAMA.git
```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.

No additional setup or installation is required.
